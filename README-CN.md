## 主要内容
本笔记基于数学系专业课程要求，涵盖以下核心模块：
* **定解问题**：波动方程、热传导方程、拉普拉斯方程的推导与物理背景。
* **经典解法**：分离变量法、行波法、积分变换法（Fourier & Laplace）。
* **特殊函数**：Legendre 多项式、Bessel 函数的性质及其应用。
* **数值分析**：结合数值解法的稳定性与收敛性讨论。
